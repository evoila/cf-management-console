T-Shirt size erstellen

Mit T-Shirt sizes können Sie die Größe der jeweiligen Vorlagen für die VMs bestimmen. Eine T-Shirt size wird bei der Erstellung eines Auftrages als Auswahl zur Verfügung gestellt. Änderung an einer T-Shirt size ziehen aktuell keine vollautomatischen Anpassung der Größen bei den existierenden VMs nach sich.
